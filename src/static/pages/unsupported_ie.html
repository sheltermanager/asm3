Internet Explorer

We've detected that you are using Internet Explorer. ASM no longer supports this browser.

This browser is old, insecure, no longer receives updates from Microsoft and does not support many modern web standards and features that ASM needs to run.

We really don't like forcing people to upgrade to something else, but it is dangerous to continue to use this browser as not only will the web increasingly not work with it, it represents a considerable security risk to your personal information and as an attack vector to your computer for viruses and malware. Please upgrade your web browser as soon as you can.

You can use any of the following standards-compliant browsers:

  • Firefox
  • Google Chrome
  • Safari
  • Microsoft Edge
